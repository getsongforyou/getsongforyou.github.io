# qing-meng
青檬，一个基础的后台模板，基于vue2 和  node

## 技术相关

vue2 + vue-cli + element-ui + axios

## 功能

目前
* 登录
* 注销
* 富文本
* 表格
* 动态权限
* 动态挂载路由
* 跨域
* 动态面包屑

待实现

* 主题


### 跨域

后台node 使用中间件cors实现跨域
